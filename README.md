# Web-API

- Spring Boot를 이용한 Web API 구축

## 개발환경

- JAVA 1.8
- Oracle DataBase 11.g

## 구현 목표

- 오라클 연동
- Web Socket 통신
- (Optional) Vue.js 적용

## 진행사항

- Oracle-JPA 연동 (06/03)
- Swagger 적용 (06/04)
- Security 적용 (06/05)
